bookTentang Website Ini

Website ini saya buat sebagai bagian dari Praktikum CSS Framework pada Mata Kuliah Desain Dan Pemrograman Web. Framework yang digunakan adalah Materialize CSS, yaitu framework berbasis Material Design dari Google yang menyediakan komponen UI modern dan responsif.

Dengan Materialize, pembuatan website menjadi lebih cepat dan terstruktur karena kita cukup menggunakan class-class yang sudah tersedia tanpa perlu menulis CSS dari nol.
