Kesimpulan:

- type="image" berfungsi sebagai tombol submit dalam bentuk gambar.

- type="file" memungkinkan pengguna mengunggah file dari komputer.

- type="hidden" digunakan untuk menyimpan data yang tidak terlihat oleh pengguna (seperti ID transaksi) tetapi ikut terkirim ke server.

Uji coba 7



Kesimpulan: Atribut min dan max membatasi nilai yang boleh diinput. Saat nilai min diubah menjadi -50 pada input range, titik awal geser (slider) akan dimulai dari angka negatif. Input number akan memvalidasi agar angka tidak keluar dari rentang saat form dikirim.

Uji coba 8




Kesimpulan:

- type="url" memvalidasi format alamat web (harus ada http://).

- type="search" memberikan kolom pencarian (biasanya ada tanda silang untuk menghapus teks).

- type="tel" dengan atribut pattern memaksa pengguna mengisi nomor telepon dengan format tertentu (misal: 123-45-678).
