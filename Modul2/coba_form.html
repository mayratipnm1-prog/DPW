Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

Kesimpulan Ujicoba:

Kesimpulan: Tombol reset berfungsi mengembalikan form ke nilai awal (default). Atribut target="_blank" akan membuka hasil pengiriman di tab baru, sedangkan target="_self" di tab yang sama. Pada method="get", data terlihat jelas di URL browser, sedangkan pada method="post", data tersembunyi di dalam paket HTTP sehingga lebih aman untuk data sensitif.
